# Инструкция по деплою на Vercel

## Способ 1: Через веб-интерфейс Vercel (Рекомендуется)

1. Перейдите на https://vercel.com
2. Войдите через GitHub (используйте тот же аккаунт, что и для репозитория)
3. Нажмите "Add New Project"
4. Выберите репозиторий `dmitrymvp/padel-yugra`
5. Vercel автоматически определит, что это Next.js проект
6. Нажмите "Deploy"
7. Через несколько минут сайт будет доступен по адресу типа: `padel-yugra.vercel.app`

## Способ 2: Через Vercel CLI

1. Установите Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. Войдите в Vercel:
   ```bash
   vercel login
   ```

3. Задеплойте проект:
   ```bash
   vercel
   ```

4. Для продакшн деплоя:
   ```bash
   vercel --prod
   ```

## После деплоя

- Сайт будет доступен по адресу, который предоставит Vercel
- Каждый push в main ветку будет автоматически деплоить новую версию
- Можно настроить кастомный домен в настройках проекта на Vercel
